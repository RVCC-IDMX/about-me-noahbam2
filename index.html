Techniques for Rendering Scenes

Raytracing

Gaming industries, Movie production studios, and practically any company who's dipping their toes into realism has most definitely used a technique for creating real world lighting called Raytracing. An article by NVidia put out a simplified way of explaining what raytracing is and what it does for the gaming and movie industry. The article sums up what it means to ray trace by stating that it basically simulates shadows, reflections, refractions, and indirect light using heavy computational power. It takes the camera (which is the view of the scene) and calculates the view ray through one pixel in the image and hits an object directly beaming into the light, which creates the image on screen. The View ray makes interactions with the 3d environment by passing through objects which might be reflective. Some rays might make it through objects; thats semi-transparent or (refractive). Some of the light rays might bounce off an object and back into the camera to produce a reflective material. All internally producing a convincing image. Raymarching

A similar topic on the process of ray tracing, there is also a much newer method known as raymarching, raymarching is a way to render entire environments procedurally within a shader way to instead of using actual geometry with quads, tris and other polygonal shapes that make up the shape of an object it can mathematically produce shapes without using those aspects, since the process is entirely math based, ray marched objects will never lose quality as you zoom into it. There are many different methods for approaching the same result with varying optimizations, but one that is called Sphere tracing is a common one. An article posted on Wikipedia states,” The SDF is evaluated for each iteration in order to be able take as large steps as possible without missing any part of the surface.” (Wikipedia Contributors 3 May 2023). To sum it up, an algorithm uses a sphere to check whether there is an object in the scene, after it hits a ray is shot from the camera and each iteration of the ray checks for when the sphere hits an object.
